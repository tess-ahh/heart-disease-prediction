#Heart Disease prediction
This repository contains my Heart Disease Prediction project built using Python and machine learning techniques.
Unlike my first repository, which was an introductory profile, this is my first complete, end-to-end project involving real data, preprocessing, model training, evaluation, and interpretation. 
The project uses the UCI Heart Disease dataset and implements a Random Forest Classifier to predict whether a patient has heart disease based on clinical features.
